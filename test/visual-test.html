VectorScope — Visual Integration Test

Tests the homography pipeline visually: each canvas shows a grid of output pixels mapped through the lead (camera color) and follower (red) sampling matrices. Overlapping grids at the center = correct focus-plane alignment.

Test 1: Lead + Follower Grid Alignment (warp ON)

Grid of output pixels mapped to source RT coordinates. Lead grid = camera color. Follower grid = red. At the image center, both grids should converge. Edge divergence = parallax (expected at non-focus depths).

Test 2: Zoom Sweep — Matrix Continuity

Lead matrix H[0][2] (horizontal translation) plotted across the full zoom range. The curve should be smooth with no discontinuities at segment boundaries (1.0x, 5.0x). Color = source camera.

Test 3: Warp ON vs OFF Comparison

Left pair: warp OFF (prewarp crop only — grids misaligned). Right pair: warp ON (homography-corrected — grids aligned at center).

Test 4: Blend Transition (UW → Main at 1.0x)

Simulates zooming from 0.85x to 1.29x. Blend bar shows cross-fade progress. Outgoing camera grid fades out as incoming fades in.
